Getting Started with HTML - Notes

- The web is a network of computers that talk to each other through server requests (HTTP)
- Major pieces of the web are:
        - Computer
        - Servers
        - HTTP
        - Internet
        - Browser

HTML
HTML is made up of text content, markup, ref to other docs, and links

Tags
Tags tell the content how to look and what to do. A few examples:
        - Image "img"
        - Anchor "a"
        - Emphasis "em"
        - Bold "b"
        - Paragraph "p"

-White space = single space
-Inline refers to the content of a single line
-Block refers to the content within a box
-Span = Inline
-Div = Block
